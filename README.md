# Warehouse Inventory System

## Project Description
A Java application designed to manage warehouse stock levels using encapsulation and modular classes.

## How to Run the Program
1. Open the terminal.
2. Compile: `javac Main.java`
3. Run: `java Main`

## Sample Output
> Item Added: Laptop (Qty: 50)
> Current Inventory: 150 units total.
